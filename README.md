# 関節角度・歩行分析

スマホで使える、手動式の関節角度計測・歩行分析アプリです。

## 関節角度

- 写真を選びます。
- 近位端、関節中心、遠位端の順に3点をタップします。
- 点はドラッグで微調整できます。
- ROM表示は `180° - 内角`、内角表示は3点そのままの角度です。
- 記録と、計測線入り画像保存ができます。

## ROM比較

- 基本肢位と最大可動の2枚の写真を選びます。
- それぞれ同じ順番で3点をタップします。
- 2枚の角度差をROMとして表示します。
- 2枚を並べた画像保存ができます。

## 歩行分析

- 歩行動画を選びます。
- 接地・離地の瞬間で一時停止し、「右接地」「左接地」「右離地」「左離地」を記録します。
- ケイデンス、ステップ時間、ストライド時間、立脚相の目安を出します。
- 距離校正をすると、歩幅もcmで記録できます。

## 注意

このアプリは診断用ではなく、記録・比較・説明の補助です。撮影方向、点の置き方、動画のフレーム位置で結果が変わります。
